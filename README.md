# Ionic project Books-Store

## Built on:

### - NestJS

### - Ionic 

### - NodeJs 

### - npm

### - MySql


## Running the app on Windows

```
$ npm i

# development
$ ionic serve
```

#### **Android:** 

```
# development
$ ionic cordova run android -l
```

#### Features:

google auth, JWT auth, infinite scroll, payPal, Stripe
